# Introduction

This is a simple bash install script for Stoat, an open source alternative for discord.

This script will download Stoat from the official GitHub repo, install it in $HOME/.local/bin/, and create a .desktop file for your application launcher.

# Usage

## Automatic download/install

Copy/paste this command to automatically download and install Stoat:

```shell
git clone https://github.com/JshKlsn/stoat_install_script && cd stoat_install_script && chmod +x stoat_install.sh && ./stoat_install.sh
```
